Hover Effects: Change styles on hover. .button:hover { background-color: blue; color: white; } Transitions: Smoothly change property values. .box { transition: transform 0.3s; } .box:hover { transform: scale(1.1); } Animations: Create keyframe animations. @keyframes fadeIn { from { opacity: 0; } to { opacity: 1; } } .fade-in { animation: fadeIn 2s; } Box Shadows: Add depth to elements. .card { box-shadow: 0 4px 8px rgba(0, 0, 0, 0.2); } Text Shadows: Create depth in text. .title { text-shadow: 2px 2px 4px rgba(0, 0, 0, 0.5); } Gradient Backgrounds: Use gradients for backgrounds. .gradient { background: linear-gradient(to right, red, blue); } Transformations: Rotate, scale, translate, and skew elements. .rotate { transform: rotate(45deg); } Opacity: Control the transparency of elements. .transparent { opacity: 0.5; } Clip-path: Create complex shapes. .clip { clip-path: circle(50%); } Flexbox and Grid: Create responsive layouts. .flex-container { display: flex; justify-content: space-between; } .grid-container { display: grid; grid-template-columns: repeat(3, 1fr); } These effects can be combined for even more dynamic and engaging designs.
